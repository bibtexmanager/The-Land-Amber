# Lithuania Booth Variants

This workspace was created without modifying the original game files in `Downloads`.

Structure:
- `00-original-snapshots` - copied originals for reference
- `01-map-challenge` - improved Lithuania map booth game
- `02-flag-pick` - visual flag selection game
- `03-photo-quiz` - image-based Lithuania quiz
- `04-booth-mix` - mixed booth arcade game
- `index.html` - landing page for all variants

GitHub Pages:
1. Upload the contents of `lithuania-booth-variants` to a repository root.
2. Enable Pages from the `main` branch and `/ (root)`.
3. The hub page will be served from `/`.

- `index.html`
- `01-map-challenge/index.html`
