Lithuania Booth Game Variants

Separate, non-destructive booth-ready variants built alongside the original files. Each game lives in its own folder and can be opened or published independently.

Originals were not edited in place. A snapshot copy of the original map game and quiz is stored in 00-original-snapshots.
Map Variant

01 Map Challenge

A better "Find Lithuania" booth game with multiple prompts, clickable map regions, streak scoring, and geography rounds instead of the same Lithuania tap every time.

Flag Variant

02 Flag Pick

Fast visual rounds for "Choose Lithuania's flag" and related Baltic flag recognition. Good for quick kiosk play and younger visitors.

Photo Variant

03 Photo Quiz

Image-led rounds using landmark, food, amber, and culture cards so the booth experience is less text-heavy and more immediately playable.

Mixed Variant

04 Booth Mix

A mixed-mode arcade build that rotates between map, flag, photo, and trivia prompts in one game loop.

Original Snapshot

00 Original Snapshots

Safe copied versions of the current original files for comparison and rollback.

This folder can also be used as a GitHub Pages site root. The main landing page is this file, and each variant is reachable by its folder path.
